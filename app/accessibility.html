Accessibility Statement

Last updated: May 2026

Our Commitment

Psychic101 is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and strive to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA and the Israeli standard IS 5568.

Current Accessibility Features

  • Semantic HTML: Pages use proper heading hierarchy, <header>, <main>, and <footer> landmarks.
  • Skip Navigation: A "Skip to content" link is provided on all pages for keyboard and screen reader users.
  • Keyboard Accessibility: All interactive elements are operable via keyboard navigation.
  • Focus Indicators: Visible focus outlines are provided for all interactive elements.
  • Color Contrast: Text and background colors meet WCAG AA contrast ratio requirements (minimum 4.5:1 for normal text).
  • Responsive Design: The site works on phones, tablets, and desktops with appropriate touch targets (minimum 48×48px).
  • Language Declaration: The lang attribute is set correctly on all pages and updates dynamically when switching between English and Hebrew.
  • RTL Support: Full right-to-left layout support for Hebrew, including proper text direction and mirrored UI elements.
  • Screen Reader Compatibility: ARIA attributes and semantic markup assist screen reader users.

Known Limitations

  • Some interactive canvas elements (ideogram drawing, dial trainer) may present challenges for screen reader users. We are working on providing alternative text descriptions.
  • Complex data visualizations in the Signal Line Analyzer may not be fully accessible to all assistive technologies.

Feedback

We welcome your feedback on the accessibility of Psychic101. If you encounter accessibility barriers, please let us know:

  • Open an issue on our GitHub repository

Evaluation

The Psychic101 accessibility evaluation process includes:

  • Self-review against WCAG 2.1 AA and IS 5568 criteria
  • Automated accessibility testing tools
  • Manual testing with keyboard navigation and screen readers
