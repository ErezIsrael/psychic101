Accessibility Statement

Last updated: May 2026

Our Commitment

Psychic101 is committed to digital accessibility.

Current Accessibility Features

  • Semantic HTML
  • Skip Navigation
  • Keyboard Accessibility
  • Focus Indicators
  • Color Contrast
  • Responsive Design
  • Language Declaration
  • RTL Support
  • Screen Reader Compatibility

Known Limitations

  • Canvas elements may challenge screen readers
  • Complex data visualizations may not be fully accessible

Feedback

Evaluation

The evaluation process includes:

  • Self-review against WCAG 2.1 AA and IS 5568
  • Automated accessibility testing tools
  • Manual testing with keyboard and screen readers
